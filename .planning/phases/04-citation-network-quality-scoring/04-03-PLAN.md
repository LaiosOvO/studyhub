1. Create `backend/app/schemas/quality.py`:
   - `QualityWeights(BaseModel)`: citations: float = 0.35, velocity: float = 0.25, impact_factor: float = 0.20, h_index: float = 0.20. Add validator ensuring sum == 1.0 (with 0.01 tolerance).
   - `QualityBreakdown(BaseModel)`: score: float (0-1), citations_norm: float, velocity_norm: float, impact_factor_norm: float, h_index_norm: float, components_available: int (how many of 4 had data).
   - `PaperWithQuality(PaperResult)`: quality: QualityBreakdown | None = None.
   - `TopPapersRequest(BaseModel)`: paper_ids: list[str] | None = None (if None, use all papers in Neo4j for given direction), n: int = 10 (1-100), direction: str | None = None.
   - `TopPapersResponse(BaseModel)`: papers: list[PaperWithQuality], total_scored: int.

2. Create `backend/app/services/citation_network/quality_scorer.py`:
   - `compute_quality_score(citation_count: int, year: int | None, impact_factor: float | None, h_index: int | None, weights: QualityWeights | None = None) -> QualityBreakdown`:
     - Calculate citation_velocity: if year and year < current_year, velocity = citation_count / (current_year - year + 1). If year is None or year >= current_year, velocity = citation_count (treat as 1 year).
     - Normalize each component to 0-1 using log scaling with caps (as in RESEARCH.md): citations: min(log10(count+1)/4.0, 1.0), velocity: min(velocity/50.0, 1.0), IF: min((if or 0)/20.0, 1.0), h_index: min((h or 0)/80.0, 1.0).
     - Count components_available (how many were not None/0 before normalization -- count citation_count > 0 as available, velocity always available if citations available, IF available if not None, h_index available if not None).
     - Weighted sum using provided weights or DEFAULT_WEIGHTS.
     - Return QualityBreakdown with all normalized values.
     - PURE FUNCTION: no side effects, no API calls.

   - `async get_author_h_index(author_name: str) -> int | None`:
     - Use pyalex Authors().search(author_name).get(per_page=1) via asyncio.to_thread.
     - Extract summary_stats.h_index from first result.
     - Return None on any error (log warning).

   - `async get_journal_impact(venue: str) -> float | None`:
     - Use pyalex Sources().search(venue).get(per_page=1) via asyncio.to_thread.
     - Extract summary_stats.2yr_mean_citedness from first result.
     - Return None on any error (log warning).

   - `async score_paper(paper: PaperResult) -> QualityBreakdown`:
     - Fetch h_index for first author (if authors list not empty) via get_author_h_index.
     - Fetch impact_factor for venue (if venue not None) via get_journal_impact.
     - Call compute_quality_score with all components.
     - Use asyncio.gather for parallel h_index + IF lookups.

   - `async score_papers_batch(papers: list[PaperResult], neo4j_client: Neo4jClient | None = None) -> list[PaperWithQuality]`:
     - Score each paper via score_paper. Process sequentially (OpenAlex rate limits).
     - If neo4j_client provided, batch update quality_score on Paper nodes in Neo4j via Cypher UNWIND: `UNWIND $updates AS u MATCH (p:Paper {s2_id: u.s2_id}) SET p.quality_score = u.score`.
     - Return list of PaperWithQuality.

   - `async get_top_papers(neo4j_client: Neo4jClient, n: int = 10) -> list[dict]`:
     - Cypher query: `MATCH (p:Paper) WHERE p.quality_score IS NOT NULL RETURN p ORDER BY p.quality_score DESC LIMIT $n`.
     - Return list of paper dicts from Neo4j.

3. Add `quality_score` column to Paper model in `backend/app/models/paper.py`:

   - Import Float from sqlalchemy.
