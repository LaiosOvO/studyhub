     - Include x-api-key header if api_key provided.
     - Parse `recommendedPapers` array, map each via `_map_s2_to_paper` (import from s2_client).
     - Add @retry decorator (tenacity, same pattern as s2_client: 3 attempts, exponential wait).
     - Handle 404 gracefully (return empty list).
   - `async discover_and_store_similar(paper_s2_id: str, neo4j_client: Neo4jClient, http_client: httpx.AsyncClient, limit: int = 20, api_key: str = "") -> list[PaperResult]`:
     - Call get_similar_papers, then batch_merge_papers for all results, then create RELATED_TO edges in Neo4j between source paper and each similar paper. Use a separate Cypher MERGE for RELATED_TO edges (not CITES): `MATCH (a:Paper {s2_id: $source_s2_id}) MATCH (b:Paper {s2_id: $target_s2_id}) MERGE (a)-[r:RELATED_TO]->(b) ON CREATE SET r.source = 's2_recommendations'`.
     - Add a `batch_merge_similarity_edges` method to Neo4jClient for this (or use existing execute_write with raw Cypher). Prefer adding a method to neo4j_client.py.

2. Create `backend/app/routers/citations.py`:
   - `router = APIRouter(prefix="/citations", tags=["citations"])`
   - **POST /citations/expand**: Accepts CitationExpansionRequest body. Gets neo4j and http_client from request.app.state. Creates S2 client from http_client + settings.s2_api_key. Calls expand_citations. Also calls discover_and_store_similar for each seed paper. Returns CitationExpansionResponse. Rate limit: 5/min (expansion is expensive). Return 503 if neo4j is None.
   - **POST /citations/expand-node/{paper_s2_id}**: Single-depth expansion for CITE-06 (manual expansion from graph view). Takes paper_s2_id path param, optional query params: budget (default 20, max 100). Creates S2 client, calls expand_citations with seed=[paper_s2_id], max_depth=1, budget_per_level=budget. Also discovers similar papers. Returns CitationExpansionResponse. Rate limit: 10/min.
   - **GET /citations/graph/{paper_id}**: Returns the Neo4j neighborhood of a paper. Takes paper_id path param, depth query param (default 1, max 3). Calls neo4j_client.get_paper_neighborhood. Returns dict with nodes and edges. Rate limit: 30/min.
   - All endpoints check `request.app.state.neo4j is not None`, return 503 if unavailable.

3. Update `backend/app/main.py`:
   - Import citations router and register it: `application.include_router(citations_router, prefix="/citations", tags=["citations"])`. Note: remove prefix from router definition if you set it on include_router, or use one or the other (not both). Be consistent -- use prefix on include_router to match existing pattern in main.py.
   - Also add `batch_merge_similarity_edges` to neo4j_client.py if not done in step 1.
