REST API endpoints (from backend/app/routers/plans.py):
```
POST   /api/v1/plans/generate  — starts plan generation, returns {workflow_id, task_id}
GET    /api/v1/plans/          — list plans (query: task_id, status, skip, limit)
GET    /api/v1/plans/{plan_id} — get single plan
PATCH  /api/v1/plans/{plan_id} — update plan fields
DELETE /api/v1/plans/{plan_id} — delete draft plan
POST   /api/v1/plans/{plan_id}/approve — approve plan for execution
```

ExperimentPlanResponse fields:
```typescript
interface ExperimentPlan {
  id: string;
  user_id: string;
  task_id: string;
  entry_type: "direction" | "paper" | "gap";
  source_paper_id: string | null;
  source_gap_index: number | null;
  title: string;
  hypothesis: string;
  method_description: string;
  baselines: Array<{name: string; paper_id?: string; metrics?: Record<string, any>}>;
  metrics: string[];
  datasets: Array<{name: string; url?: string; size?: string; license?: string}>;
  technical_roadmap: Array<{step: number; description: string}>;
  code_skeleton: string | null;
  feasibility: {
    compute_requirements: number;
    data_availability: number;
    expected_improvement: number;
    difficulty: number;
    overall: number;
    explanation: string;
  } | null;
  data_strategy: "open_source" | "own_data" | "hybrid";
  status: "draft" | "approved" | "executing" | "completed";
  created_at: string;
  updated_at: string | null;
}
```
