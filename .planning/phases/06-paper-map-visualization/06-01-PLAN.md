2. Create `apps/web/src/stores/paper-map-store.ts` -- Zustand store shared across all views:
   ```typescript
   interface PaperMapState {
     readonly activeView: 'graph' | 'topic' | 'timeline';
     readonly selectedPaperId: string | null;
     readonly filters: {
       readonly yearRange: [number, number] | null;
       readonly methodTypes: readonly string[];
       readonly qualityThreshold: number;
     };
     readonly readingListPaperIds: readonly string[];
     readonly setActiveView: (view: PaperMapState['activeView']) => void;
     readonly selectPaper: (id: string | null) => void;
     readonly setFilters: (filters: Partial<PaperMapState['filters']>) => void;
     readonly toggleReadingListPaper: (paperId: string) => void;
   }
   ```
   Default filters: yearRange null (no filter), methodTypes [] (all), qualityThreshold 0 (all).
   toggleReadingListPaper: adds if not present, removes if present (immutable -- create new array).

3. Create `apps/web/src/lib/graph-transforms.ts`:
   - `toReactFlowGraph(apiResponse)` -- transforms Neo4j neighborhood response `{nodes, edges}` into React Flow `{nodes: Node[], edges: Edge[]}`.
     - Node positions initialized to {x: 0, y: 0} (d3-force computes real positions).
     - Node type set to 'paper'.
     - Node data includes: title, citationCount, qualityScore, year, clusterId (extracted from node properties; use 'default' cluster if no cluster_id).
     - Edge styling: CITES edges solid gray (#6b7280), RELATED_TO edges dashed orange (#d97706) with animated=true.
   - `toDeckGlPoints(apiResponse)` -- stub for Plan 02 (returns paper objects with x=0, y=0).
   - `toTimelineItems(apiResponse)` -- stub for Plan 02 (returns vis-timeline item shape).
   All functions MUST be pure (no side effects, no mutation).

4. Create `apps/web/src/components/paper-map/hooks/useGraphData.ts`:
   - Accepts taskId string parameter.
   - Fetches deep research task detail from `/api/deep-research/tasks/{taskId}` to get paper list.
   - Fetches graph from `/api/citations/graph/{seedPaperId}?depth=2` for the task's seed papers.
   - Returns `{ nodes, edges, isLoading, error }` after running toReactFlowGraph.
   - Use useEffect for fetch on mount. Store raw API response in a ref for other views to transform.

5. Create `apps/web/src/components/paper-map/hooks/useFilteredData.ts`:
   - Reads filters from usePaperMapStore.
   - Accepts nodes and edges arrays.
   - Returns filtered nodes and edges (remove nodes outside yearRange, below qualityThreshold, not matching methodTypes; remove edges where source or target was filtered out).
   - All filtering is client-side, immutable (new arrays).

   - Circular div sized by citation count: `Math.max(30, Math.min(80, Math.sqrt(citationCount) * 5))` px.
   - Background color from cluster: use d3-scale-chromatic schemeTableau10 indexed by clusterId.
   - Display citation count number inside circle.
   - Tooltip (title attribute) showing paper title.
   - Hidden Handle elements (target top, source bottom) for edge connections.
   - Wrap with `memo()` to prevent re-render on unrelated changes.
   - CRITICAL: Define `const nodeTypes = { paper: PaperNode }` OUTSIDE the component that uses it (in CitationGraph.tsx) to prevent React Flow re-mount bug.

2. Create `CitationEdge.tsx` -- optional custom edge (or use default React Flow edges with style config in graph-transforms). If using default edges, skip this file and apply styling in toReactFlowGraph edge mapping. Decision: use default edges with style -- simpler, fewer files.

3. Create `useForceLayout.ts` -- D3 force simulation hook:
   - Accepts initial nodes and edges from React Flow format.
   - Runs forceSimulation with: forceManyBody (strength -300), forceLink (distance 100, id by node.id), forceCenter(0,0), forceCollide(radius based on node citation count).
   - Updates node positions via setNodes callback. To avoid jank with 200+ nodes, only call setNodes every 3rd tick using a counter.
   - Stops simulation when alpha < alphaMin (layout settled).
   - Cleanup: return simulation.stop() from useEffect.
   - Re-runs when initialNodes/initialEdges identity changes (use JSON.stringify length check or a version counter to avoid unnecessary restarts).

4. Create `CitationGraph.tsx` -- main graph view:
   - Uses ReactFlow component from @xyflow/react with: fitView, zoomOnScroll, panOnDrag, nodesDraggable, minZoom=0.1, maxZoom=3.
   - Receives nodes/edges from useFilteredData (passed from MapView).
   - Applies useForceLayout to position nodes.
   - On node click: calls usePaperMapStore selectPaper with the node's paper_id.
   - nodeTypes defined OUTSIDE component: `const nodeTypes = { paper: PaperNode }` at module level.
   - Import ReactFlow CSS: `import '@xyflow/react/dist/style.css'`.

5. Create `PaperDetailPanel.tsx` -- sidebar panel:
   - Reads selectedPaperId from usePaperMapStore.
   - If selectedPaperId is null, render nothing.
   - Fetches quality breakdown from `/api/citations/paper/{selectedPaperId}/quality` when selectedPaperId changes.
   - Displays: paper title, year, citation count, quality score with breakdown bars, methods (from node data or task config), abstract snippet.
   - Close button that calls selectPaper(null).
   - Positioned as a fixed right sidebar (w-80) overlaying the graph.
   - Bilingual labels using next-intl useTranslations('paperMap').

6. Create `MapView.tsx` -- view switcher container:
   - Tab bar with three tabs: "Citation Graph" / "Topic Map" / "Timeline" reading activeView from store.

   - Passes nodes/edges from useGraphData through useFilteredData to CitationGraph.
