```python
class BasePaperClient(ABC):
    def __init__(self, http_client: httpx.AsyncClient) -> None:
        self._client = http_client

    @abstractmethod
    async def search_keywords(self, query: str, limit: int = 25) -> list[PaperResult]: ...
    @abstractmethod
    async def search_doi(self, doi: str) -> PaperResult | None: ...
    @abstractmethod
    async def search_author(self, author: str, limit: int = 25) -> list[PaperResult]: ...
```

1. Create `backend/app/services/paper_search/cnki_client.py`:
   - Define `CnkiCaptchaError(Exception)` and `CnkiSelectorMismatchError(Exception)`.
   - `CnkiClient` inherits from `BasePaperClient`.
   - Constructor takes `http_client: httpx.AsyncClient` and `browser_pool: BrowserPool`. Store pool as `self._pool`.
   - **CSS selectors as module-level constants** (not hardcoded in methods) following RESEARCH.md `CNKI_SELECTORS` and `CNKI_SELECTORS_FALLBACK`. This makes maintenance easy when CNKI changes DOM.
   - `search_keywords(query, limit=25)`:
     - URL: `https://kns.cnki.net/kns8s/defaultresult/index?kw={query}`
     - Use `self._pool.get_context("cnki.net")` to get a browser context.
     - Navigate with `wait_until="networkidle"`, timeout 30000ms.
     - Call `_check_captcha(page)` -- if `.verify-img-panel` or redirect to `vericode` detected, raise `CnkiCaptchaError`.
     - Get `page.content()`, parse with `_parse_results(html, limit)`.
     - If page loaded (200) but 0 results parsed, log a `SELECTOR_MISMATCH` warning (DOM structure may have changed).
     - Always close page in `finally` block.
     - Rate limit: `await asyncio.sleep(4.0)` after each request.
   - `search_doi(doi)`: Return None (CNKI does not support DOI lookup in public interface).
   - `search_author(author, limit=25)`: Same as keywords but URL uses `&korder=AU` parameter.
   - `_parse_results(html, limit)`: Parse with BeautifulSoup + lxml. Use primary selectors first, fall back to fallback selectors if primary yields 0 results. For each result row, extract: title, authors (list), venue, year (from date td), citation_count (from quote td). Set `language="zh"`, `sources=[PaperSource.CNKI]`. Extract cnki_id from the title link href if possible. Return `list[PaperResult]` capped at `limit`.
   - `_check_captcha(page)`: Check for CAPTCHA indicators (`.verify-img-panel`, URL contains `vericode`). If detected, raise `CnkiCaptchaError("CAPTCHA verification required")`.
   - Helper methods: `_safe_text(item, selector) -> str | None`, `_extract_year(item) -> int | None`, `_extract_citations(item) -> int`.
