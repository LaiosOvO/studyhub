       - Extract research fields from career description
       - Extract honors from honors section
       - Extract education history
     - Return `ScholarCreate` with normalized fields, or None if page not found / parsing fails
     - Log warnings on parsing failures, never raise -- return None
   - Use CSS selectors as module-level constants (same pattern as CNKI/Wanfang scrapers per STATE.md decisions: "CSS selectors as module-level constants with fallback dicts")
